#  Hospital Admissions & Discharge Analytics Dashboard

##  Project Overview
This project presents an interactive **Power BI dashboard** designed to analyze hospital **admissions, discharges, and bed occupancy trends**.  
The dashboard transforms raw healthcare data into **actionable insights** to support hospital **capacity planning, operational efficiency, and decision-making**.

The project combines **data modeling, DAX calculations, and visual storytelling**, with a clean layout and consistent theme design.

---

## Objectives
- Analyze patient **admission and discharge patterns**
- Monitor **bed occupancy rate (%)**
- Compare **weekday vs weekend admissions**
- Identify trends by **department, diagnosis, and time**
- Demonstrate advanced **Power BI & DAX skills**

---

##  Dashboard Features
- **Hourly admissions table** by weekday
- **Monthly trend analysis** (Admissions vs Discharges)
- **Discharge analysis by Department**
- **Discharge analysis by Diagnosis**
- **Weekday vs Weekend Admissions (%)**
- Dynamic measure switching (Value vs % of Grand Total)

---

##  Key DAX Concepts Used
- `CALCULATE`
- `DIVIDE`
- `SWITCH`
- `SELECTEDVALUE`
- `ALLSELECTED`
- Time intelligence with calendar table
- Percentage of grand total calculations
